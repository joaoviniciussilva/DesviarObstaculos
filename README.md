Projeto prático desenvolvido durante o curso "Complete C# Unity 3D Game Development" (Udemy).
Este projeto faz parte do meu aprendizado no desenvolvimento de jogos 3D. Utilizando a Unity 6 
explorei a criação de um jogo de desvio de obstaculos focado em mecanicas de física e lógica de 
script.

Aprendizados e Implementaçoes:
Durante o desenvolvimento e seguindo a metodologia do curso, implementei: 
- Scripts limpos e modulares para controle de jogador e deteccao de colisões;
- Uso de RigidBody e Velocity para movimentação fluida em ambiente 3D;
- Uso estratégico de Start(), Update() e FixedUpdate() para otimizaçao de performance;
- Criaçao de obstaculos reutilizaveis;

Tecnologia e Ferramentas: 
- Engine: Unity 6;
- Linguagem: C#;
- IDE: Visual Studio/Vs Code;
